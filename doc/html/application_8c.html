A memory block is allocated for the read applications, and is returned by the argument 'apps'. The returned integer is the number of applications read. If the integer is negative, one of the following errores ocurred: EAR_ALLOC_ERROR, EAR_READ_ERROR or EAR_FILE_NOT_FOUND.
