int getenv_ear_local_id ( )

int getenv_ear_num_nodes ( )

Reads the number of nodes from the EAR_NUM_NODES environment variable and returns it.

If it's not set, it tries to see if SLURM_STEP_NUM_NODES is set and returns it. Otherwise returns 1.

unsigned long getenv_ear_p_state ( )

Reads the EAR_P_STATE environment variable's value and returns it if it's set, otherwise returns the default value for the current power policy.

getenv_ear_power_policy() is automatically called when using this function if it had not been called before.

double getenv_ear_performance_accuracy ( )

Reads the PERFORMANCE_ACCURACY environment variable's value and returns it if it's set, otherwise returns returns the default value.

int getenv_ear_power_policy ( )

Checks what the EAR_POWER_POLICY environment variable is set to.

Returns 0 if it's not set or it's set to MIN_ENERGY_TO_SOLUTION or min_energy_to_solution, 1 if it's set to MIN_TIME_TO_SOLUTION or min_time_to_solution and 2 if it's set to MONITORING_ONLY or monitoring_only.

double getenv_ear_power_policy_th ( )

Reads the power threshold for the power policy currently set.

If none is specified, it sets itself to the default for that power policy. getenv_ear_power_policy() is automatically called when using this function if it had not been called before.
